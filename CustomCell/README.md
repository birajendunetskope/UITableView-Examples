CustomCell
====================

条件
---------------
 * Xcode 4.2
 * iOS SDK 5.1
 * Objective-C

概要
---------------
 * UITableViewCell をカスタマイズしたサンプル
 * ダミーの配列を用意してテーブルセルにそれぞれの値を設置
 * テキスト要素を drawRect で描画して高速化
 * 編集ボタンを押した時のレイアウトテスト
