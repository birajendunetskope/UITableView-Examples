CoreData
====================

条件
---------------
 * Xcode 4.5.2
 * iOS SDK 6.0
 * Objective-C
 * iOS 5.1以上

概要
---------------
 * Core Data（永続化フレームワーク）を使ったサンプル
 * Automatic Reference Counting （自動リファレンスカウンタ）を使用
