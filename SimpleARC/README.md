SimpleARC

条件
---------------
 * Xcode 4.5.2
 * iOS SDK 6.0
 * Objective-C
 * iOS 5.1以上

---------------
 * Automatic Reference Counting（自動リファレンスカウンタ）を使ったサンプル
 * ダミーの配列を用意してテーブルセルの textLabel へ表示
